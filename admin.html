NOIPRO License Admin (Lite)

1. Nhập thông tin giấy phép

Copy từ phần mềm NOIPRO (nút "Copy mã máy").
Đây là mã bạn gửi cho giáo viên để nhập vào ô "Mã kích hoạt".
Ví dụ: 2026-01-20. Nếu muốn trọn đời có thể đặt rất xa (2099-12-31).
Có thể chỉnh về "revoked" nếu muốn tạm ngưng license.
Phải trùng với biến môi trường ADMIN_SECRET trong Vercel.
(Không gõ đúng thì server sẽ báo "Sai ADMIN_SECRET").

2. JSON (dùng để lưu trong data/licenses.json)

Đây chỉ là phần xem trước. Khi bấm "Ghi license lên GitHub", trang sẽ gọi API /api/add để tự động thêm/cập nhật bản ghi vào data/licenses.json.
